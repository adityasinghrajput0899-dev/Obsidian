# SOP for general A+ Content Annotation

## 0. Contents

## 1. Objective
The purpose of this SOP is to annotate the rubrics for A+ Content. The results of the annotation will be used to train A+ Quality Score 2.0 model, which will serve as the foundation for downstream treatments including tiered CX treatment in detail page, customized recommendations to eliminate defects.

## 2. Version Table
Version Details Updated by Updated Date
Version 1 Basic SOP creation carriesg 2/24/2025
Version 1.1 Text readability clearification carriesg 5/19/2025

## 3. Review Instruction
To ensure accuracy, the annotation task will be handled by 3-5 different individuals independently, without seeing result from other teammates.

## 4. Checks
As part of the annotation for general A+ content quality, we conduct checks across the following 7 parameters:
● Error-free text
● Defect-free image
● Redundancy-free
● Rich product feature
● Effective module usage
● Style consistency
● Image authenticity
● Text readability

## 4.1 REVIEW THE TEXT WITHIN THE A+ CONTENT
Review the quality of the text, which includes both freeform text and embedded text. Note, the text can have multiple defects.
The common defects include:
A B C D
1 Quality trait Rubrics Reason code If no defect
Grammar and spelling
1. Grammatical errors
A+ Content - Poor A+ Content -
2 2. Spelling errors
Grammar Correct Grammar
3. Awkward language (eg. translationese, incomplete sentence)
4. Punctuation, cap, spacing issues
Error-free text
Text scannability
1. Consecutive 8+ lines of text
A+ Content - Not A+ Content -
3 2. Keyword stuffing (Keyword density >40% in any paragraph, please scannable Scannable Text exclude cases when having multiple keywords is contextually necessary)

● Incomplete sentences: Penguin pattern socks— made of , strong and durable material for long time use,christmas socks.
● Translationese: St patricks day party favors: good to your friends, they will like it very much.
● Wrong Cap: Outdoor solar light: solar powered light — the lights are powered solar energy. no external electricity required.
