- The example code snippet provided in the wiki shows how to calculate the score based on the agreement between the auditor and verifier for both attribute and rationale:
```javascript
// ON SUBMIT SCRIPT
// Calculate score based on agreement between auditor and verifier for both attribute and rationale
const score = (output.auditor_attribute_response.toString() === output.verifier_attribute_response.toString() && 
              output.auditor_rationale_response.toString() === output.verifier_rationale_response.toString()) ? 100 : 0;
const finalOutput = {
  ...output,
  score: score
};
return finalOutput;
```
This code snippet suggests that the score is calculated as follows:

- If the auditor's attribute response matches the verifier's attribute response, and the auditor's rationale response matches the verifier's rationale response, the score is set to 100.
- Otherwise, the score is set to 0.

This score calculation logic is intended to be used for the normalization system, but it's not clear from the wiki if this is the only system that uses this logic.
